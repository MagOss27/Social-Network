# Rede Social - WebSocket em Tempo Real

Este projeto é uma rede social simples com comunicação em tempo real, utilizando WebSockets. Ele permite que os usuários enviem e recebam mensagens instantaneamente, sem a necessidade de recarregar a página.

## Funcionalidades

- **Cadastro de Usuário**: Registre um nome de usuário único para interação.
- **Mensagens em Tempo Real**: Troca de mensagens instantâneas e envio de imagens entre usuários conectados.
- **Identificação Visual**: Cada usuário é identificado por uma cor única.
- **Hospedagem no Render**: O servidor WebSocket é hospedado na plataforma Render.

## Tecnologias

- **Frontend**: HTML, CSS, JavaScript.
- **Backend**: Node.js com WebSocket (ws).
- **Hospedagem**: Render.

## Como Rodar Localmente

1. Clone o repositório:

   ```bash
   git clone https://github.com/seu-usuario/rede-social.git
   cd rede-social

2. Instale as dependências:
   ```bash
   npm install

3. Inicie o servidor:

   ```bash
   node server.js

4. Abra o arquivo index.html no navegador para usar a aplicação localmente.
